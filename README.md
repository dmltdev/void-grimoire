# Omniclode Plugin

Omniclaude is a composable workflow for your coding agents, built on top of a set of composable "skills" and some initial instructions that make sure your agent uses them.

## Installation

**Note:** Installation differs by platform.

### Claude Code Official Marketplace

Omniclaude is available via the [official Claude plugin marketplace](https://claude.com/plugins/omniclaude)

Install the plugin from Claude marketplace:

```bash
/plugin install omniclaude@claude-plugins-official
```

### Claude Code

In Claude Code, register the marketplace first:

```bash
/plugin marketplace add dmltdev/omniclaude-marketplace
```

Then install the plugin from this marketplace:

```bash
/plugin install dmltdev@omniclaude-marketplace
```
